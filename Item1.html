Integrating Languages Project

In this project I combined both C++ and Python to create a mini-calculator.

This was a school project that enables the user to compute a number in several ways. As seen above, it begins with a set of options which you can choose from. Then it follows up with requesting a number and out-puts a multiplication table to the tenth multiple. First I demonstrate the multiplication table with the value 2, then I demonstrate the "Double Value" option. This second option takes an input value - the number 2 in this case - and doubles it.
